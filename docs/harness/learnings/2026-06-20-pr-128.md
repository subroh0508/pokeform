## ⚠️ Problem（詰まった点・制約）

- **archive 退避時のリンク方向が 4 種類**（active→archive で `./N`→`./archive/N`、archive 内 sibling は `./N` 維持、archive→active は `./N`→`../N`、archive 内から外部へは深さ +1）に分岐し、機械置換だけでは取り違えやすい。1 つでも誤ると dangling になる。
- **bare-number の ADR 参照（リンクでない `// ADR 0034`）が src `.ts` に残る**。[[adr]] は「`.ts` まで `git grep -n NNNN` で素の番号参照も追従」と求めるが、これらはコードの現状記述で Phase 2 でコードと共に追従すべきもの。「dangling リンク」と「bare-number のコード内記述」の追従タイミングの線引きが phase doc/rule から一意に読み取りにくかった（独立レビューで Phase 2/3 スコープと確認して解消）。
- **macOS `realpath -m` 非対応**で最初の dangling 検証が全件誤検出した。クロスプラットフォームを考えない shell ワンライナーは検証自体を誤らせる。
