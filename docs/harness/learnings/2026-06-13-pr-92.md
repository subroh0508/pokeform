## ⚠️ Problem（詰まった点・制約）

- **`noUncheckedIndexedAccess` + カバレッジ100% と正規表現キャプチャの相性**。`/...(\d+)/.exec(s)` の `m[1]` は
  `string | undefined` になり、`m[1] ?? ""` で逃がすと **到達不能な dead branch** が生まれてブランチ網羅が
  落ちる。`Number(m[1])`（Number は undefined も受ける）/ `.exec()?.[1] ?? ""`（exec が null になり得る箇所のみ
  ?? が到達可能）/ `indexOf` + `slice`（添字を避ける）で型安全 + 網羅可能に回避した。試行3回ぶんの摩擦。
- **Biome 2.4 が `__fixtures__/*.html`（実 Serebii 切片）を lint して verify を赤化した**。void 要素の閉じタグ・
  空 anchor（a11y）等で落ちる。`files.includes` の folder ignore 構文でも詰まり、`!**/__fixtures__/**` は警告
  `useBiomeIgnoreFolder`、`!**/__fixtures__/`（末尾スラッシュ）は無効化されず再び lint、**`!**/__fixtures__`
  （スラッシュ無し）が正解**だった。
- **phase doc のレイアウト指定（`src/codegen/serebii`）と `testing.md` の方針（除外パスに純関数を置かない）が
  表面的に衝突した**。codegen は既定でカバレッジ除外のため、除外を `src/codegen/*.ts`（トップレベルのみ）へ
  狭めて serebii サブツリーだけ強制対象化する判断が要った（rule にも明文化）。
- **Node エンジン警告が出続ける**。cheerio 追加で `Unsupported engine: wanted node>=24 (current v23.11.0)` が
  毎コマンド表示される。verify/CI は緑だが、ローカル Node が package の要求と乖離している。
