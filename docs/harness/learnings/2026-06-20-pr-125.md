## ⚠️ Problem（詰まった点・制約）

- ブランチ名 `harness/plan-04-...` が実体（04 分割 + 05/06 新設 + 03 整理）と乖離。GitHub のブランチリネームが open PR を追従せず close を招くためリネームを見送り、**名と実体がドリフト**した（PR 本文で補足はしたが命名規約上は不正確）。
