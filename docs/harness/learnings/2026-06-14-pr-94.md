## ⚠️ Problem（詰まった点・制約）

- **macOS BSD grep が latin-1 バイトを binary 扱いし誤動作**（`grep href` が空を返す）。DOM 調査が一時停滞。`python3` / `LC_ALL=C` でも BSD grep は不安定で、結局 python での解析に切替えた。
- **phase doc の HTML 構造仮定が実ページとズレ**ていた（メガ先リンク / 3 セクション）。doc は実ページ未取得時の推測で書かれており、実装者が実証で補正する前提だった。
- **実ページ調査中の python 範囲スライス誤りで berry 件数を一時 49 と誤認**（実際は Berries 28 + Miscellaneous Items 21 の別セクション）。cheerio パーサ側の正しい挙動（28）を「バグ」と一瞬誤判定し、不要な複数テーブル走査の動機づけにした。
- **main への auto-merge が権限分類器にブロック**された。worker 指示書は明示許可していたが分類器はそれを参照できず、ユーザー確認を要した（1 往復のオーバーヘッド）。
- **`.gitignore` タスクが実は冗長**（`data/raw` が `data/raw/serebii/` を前方一致で既にカバー）。phase doc のタスク列挙を額面どおり実装すると重複エントリを生む懸念があった。
